# Cas-Designer

> Web tool from the CRISPR RGEN Tools suite (same group as Cas-OFFinder) for choosing Cas9/Cas12a target sites with out-of-frame and off-target scoring.

## Example workflow

1. Enter a gene or sequence and select the target genome and enzyme.
2. Review candidate target sites with out-of-frame and off-target scores.
